# 📚 Study Reminder Bot
`Análise e Desenvolvimento de Sistemas - PUC Minas`
`Projeto pessoal`
`1º semestre/2026`

O Study Reminder Bot é um bot automatizado desenvolvido para transformar a rotina de grupos universitários que precisam acompanhar prazos, atividades e avaliações. O projeto surge para solucionar o problema do acesso passivo a plataformas como o Canvas, enviando lembretes automáticos diretamente no WhatsApp do grupo com todas as informações necessárias para não perder nenhum prazo.
